# amis_helper

方便部署amis的小玩意, 适用于fastapi(flask可能改改也行)

部署amis代码后访问"/amis/pages/"+你设置的路径(如果你设置的是/打头, 会去掉前缀)

具体的配置看一下src/conf吧

如果打不开就换个cdn
